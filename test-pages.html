Eco-Friendly Store - Page Test

This page tests if all the main pages are accessible and working properly.

Main Pages

  • Home Page (index.html)
  • Authentication (auth.html)
  • Shopping Cart (cart.html)
  • Checkout (checkout.html)
  • User Profile (profile.html)
  • Orders (orders.html)
  • Bill (bill.html)

JavaScript Files

  • auth.js - Authentication logic
  • cart.js - Shopping cart functionality
  • checkout.js - Checkout process
  • orders.js - Order management
  • product-details.js - Product display
  • profile.js - User profile management
  • bill.js - Bill generation
  • server.js - Email server

Test Instructions

  1. Click on each page link to verify they load properly
  2. Check that navigation between pages works
  3. Verify that JavaScript functionality works on each page
  4. Test the authentication flow (sign up/sign in)
  5. Test adding products to cart
  6. Test the checkout process

Important Notes:

  • Firebase configuration needs to be updated with your own project details
  • Email credentials need to be set in environment variables
  • Product images need to be uploaded to Firebase Storage
  • Firestore database needs to be set up with proper security rules
